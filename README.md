# 🛍️ Pepperfry Sales Data Analysis

## 📌 Project Overview

This project analyzes Pepperfry sales data using Python to uncover customer purchasing behavior, product performance, and sales trends.

The analysis includes:

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Sales Trend Analysis
- Category-wise Performance
- Customer Insights
- Data Visualization

---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

```
Pepperfry-Sales-Analysis/

```

## 🚀 How to Run

1. Clone the repository

```
git clone https://github.com/yourusername/Pepperfry-Sales-Analysis.git
```

2. Install dependencies

```
pip install -r requirements.txt
```

3. Open

```
Pepperfry.ipynb
```

using Jupyter Notebook.
